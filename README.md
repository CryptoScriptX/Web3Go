## Setuping
### wallets.txt
- private_key or mnemonic
```
0x0000000000000000
crypto script
...
```
### proxies.txt
- format `login:password@ip:port`

# Run

#### Install requirements
```sh
pip install -r requirements.txt
```
#### Run Script
```sh
python3 main.py
```

## Features

- Auto Sign in Account
- Auto Claim Every Day (UTC)
